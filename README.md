## カリキュラム外の技術
React

## アプリケーションの実行手順
```
$ git clone git@github.com:mysthkr/stockgpt-back.git
$ cd stock-gpt
$ bundle install
$ yarn install
$ rails db:create && rails db:migrate
$ rails s 
```
